# study-tools

> Custom-built tools to aid studying — utilities you've created to make learning more efficient.

## Purpose

Holds custom tools you've built to support the learning process. These are more substantial than one-off scripts — they're reusable utilities designed to help with studying.

## What Belongs Here

- Flashcard generators
- Note formatting tools
- Data processing utilities for course materials
- Any custom application built to aid studying

## What Does NOT Belong Here

- One-off scripts (those go in [automation-scripts](../automation-scripts/README.md))
