# Course Resources (STOR155)

> Instructor-provided materials for STOR155 — notes in various formats, slides, and reference spreadsheets.

---

## Purpose

Central location for all instructor-provided resources for STOR155. This course provides notes in multiple formats (detailed, blank, and partially filled) to support different study approaches.

## What Belongs Here

- Lecture notes (in various completion levels)
- Lecture slides and presentations
- Reference spreadsheets used in class

## Structure

```
Course Resources/
├── Detailed Notes/                  ← Fully completed instructor notes
├── Notes_Blank/                     ← Blank note templates for in-class filling
├── Notes_Partial/                   ← Partially filled notes
├── Slides/                          ← Lecture slides
└── Spreadsheets used:referenced/    ← Data spreadsheets used in examples
```
