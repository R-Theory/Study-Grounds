# Quiz Resources (COMP110)

> Materials for quiz preparation in COMP110, organized by quiz number.

## Purpose

Stores quiz-related materials — study guides, practice problems, and review resources — organized by individual quiz.

## What Belongs Here

- Quiz study guides and review materials
- Practice quiz questions
- One subdirectory per quiz (e.g., `Quiz 1/`)

## Structure

```
Quiz Resources/
└── Quiz 1/     ← Materials for Quiz 1
```
