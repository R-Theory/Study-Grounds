# Exams (PSYC101)

> Exam-related materials and resources for PSYC101.

## Purpose

Stores exam preparation materials, practice exams, and exam-related resources for PSYC101, organized by exam.

## What Belongs Here

- Practice exams and answer keys
- Exam study guides provided by the instructor
- One subdirectory per exam (e.g., `Exam I Resources/`)

## Structure

```
Exams/
└── Exam I Resources/    ← Materials for Exam I
```
