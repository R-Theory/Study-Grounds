# Mindmaps (PSYC101)

> Visual concept maps and diagrams for organizing PSYC101 topics and their relationships.

## Purpose

Stores mindmaps and visual diagrams that map out relationships between psychological concepts, theories, and key figures. These visual aids help connect ideas across chapters.

## What Belongs Here

- Mindmap files (image, PDF, or mindmap tool exports)
- Concept relationship diagrams
- Visual topic summaries
