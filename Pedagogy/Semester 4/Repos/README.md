# Repos

> Container for all code repositories related to this semester's coursework.

## Purpose

This directory separates **course-assigned repositories** from **personal experimental projects** related to the semester. Both types live here, but in distinct subdirectories.

## What Belongs Here

- `course-repos/` — Repositories with instructor-assigned work
- `sandbox-repos/` — Personal experimental projects inspired by coursework

## Structure

```
Repos/
├── course-repos/       ← Assigned coursework
└── sandbox-repos/      ← Personal experiments
```
