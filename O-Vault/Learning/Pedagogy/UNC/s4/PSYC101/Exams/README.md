# Exams (PSYC101)

> Exam preparation directories for PSYC101, one per exam.

## Purpose

Container for all exam-specific preparation materials for PSYC101. Each exam gets its own subdirectory following the standard study structure.

## What Belongs Here

- One subdirectory per exam (`Exam I/`, `Exam II/`, `Final/`)
- Each following the standard pattern: master study guide + notes/ + resources/
