# notes (PSYC101 Exam I)

> Chapter-by-chapter study notes covering the material tested on PSYC101 Exam I.

## Purpose

Detailed, chapter-level notes covering all material for Exam I. Each note is a thorough study reference for a single chapter, written in your own words.

## What Belongs Here

- One note per chapter covered on this exam
- Detailed explanations, definitions, key concepts, and examples
- Content written in your own words (not copy-pasted from textbooks)

## What Does NOT Belong Here

- Flashcards or Anki imports (those go in `resources/my-resources/`)
- Notes for chapters not on this exam

## Naming Conventions

- `Chapter #.md` (e.g., `Chapter 1.md`, `Chapter 2.md`)
