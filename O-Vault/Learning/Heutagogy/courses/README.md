# courses (Heutagogy Vault)

> Vault notes for self-selected courses — your notes and annotations from courses you chose to take independently.

## Purpose

Contains your personal notes from self-selected courses. The actual course code and notebooks live in the repo root at `Heutagogy/courses/` — this directory holds your understanding and annotations.

## What Belongs Here

- Course notes and summaries
- Concept breakdowns written in your own words
- Annotations on course material
