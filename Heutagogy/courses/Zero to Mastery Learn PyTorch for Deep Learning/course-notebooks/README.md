# course-notebooks

> Original course-provided notebooks — the unmodified reference material from the Zero to Mastery PyTorch course.

## Purpose

Contains the **original, unmodified notebooks** provided by the course. These serve as reference material and should not be edited directly — do your work in the [Notebooks](../Notebooks/README.md) directory instead.

## What Belongs Here

- Original Jupyter notebooks from the course repository
- Unmodified reference files

## What Does NOT Belong Here

- Your own modified or experimental notebooks (those go in [Notebooks](../Notebooks/README.md))
