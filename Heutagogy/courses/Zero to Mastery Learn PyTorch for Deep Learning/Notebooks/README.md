# Notebooks

> Your working Jupyter notebooks for the PyTorch course — where you code along and experiment.

## Purpose

Contains **your own working notebooks** as you progress through the Zero to Mastery PyTorch course. These are your implementations, experiments, and practice — not the original course-provided files.

## What Belongs Here

- Jupyter notebooks you've created or modified while following the course
- Experimental notebooks testing concepts from the course

## What Does NOT Belong Here

- Original course-provided notebooks (those go in [course-notebooks](../course-notebooks/README.md))
